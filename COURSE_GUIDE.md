## 5-Day VHS Intensive Course Guide

**Instructor:** Chiaoya Chang  
**Course Level:** Beginner to Intermediate

## Course Overview

This intensive 5-day course teaches participants how to extract, analyze, and visualize data using SQL and Python. By combining theoretical foundations with practical exercises, participants will build a complete data analysis workflow from data extraction through decision-making.

### Learning Philosophy
- **30% Theory:** Conceptual foundations and best practices
- **60% Practice:** Hands-on exercises with real datasets
- **10% Discussion:** Q&A, challenges, and peer learning

## How to Use These Materials

### For Instructors

**Before Each Session:**
1. Review the PowerPoint slides to refresh key concepts
2. Work through the complete Jupyter Notebook to familiarize yourself with examples
3. Prepare the exercise notebook and sample solutions
4. Test all code snippets in your environment

**During the Session:**
- Present concepts using the professional slide deck
- Live code along using the complete notebooks
- Have participants work through exercises
- Share solutions and discuss alternatives

**After Each Session:**
- Collect participant notebooks and feedback
- Note any questions or areas of confusion
- Prepare supplementary materials if needed

### For Participants

**Before Class:**
- Install Python 3.9+ and Jupyter Notebook
- Install required packages: pandas, numpy, matplotlib, seaborn, sqlite3

**During Class:**
- Follow along with the complete notebooks
- Run code examples in your own environment
- Complete the exercises with instructor support

**After Class:**
- Review your exercise solutions
- Explore variations and modifications
- Prepare questions for the next session

## Troubleshooting Common Issues

### Import Errors
If you see `ModuleNotFoundError`:
```bash
pip install pandas numpy matplotlib seaborn
```

### Jupyter Not Found
Install Jupyter and required packages:
```bash
pip install jupyter
jupyter notebook
```

### Database Connection Issues
Ensure sqlite3 is installed (comes with Python 3):
```python
import sqlite3
print(sqlite3.sqlite_version)
```
